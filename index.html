Resizing image with JavaScript / TypeScript by using Canvas

Demonstration of the diffrent methods and process with easy to understand code

Resizing image have two main steps:

  • Resizing exist image
  • Producing new image

This guide is dealing mainly the first step, the second step, is the same for all the methods:

  • You have a canvas with the dimensions of the target image
  • you draw the image on it in the exact size of the canvas
  • use a method to output base64 image.
    let resizedBase64image = outputCanvas.toDataURL('image/jpeg', 0.85);

Method 2 - Direct resize from original to target, using single canvas, and one resizing step

This method will result in aliased image in case the source image is much bigger then the target image dimensions. If the original images aren't that big compared to the target images, or you dont mind having some aliases, u may use this method which is much simpler.



Method 3 - Multiple resizing steps, using a loop on the same canvas

This method re-use a single canvas to do the resizes on it, each time the resized image is painted on the top of the exist bigger image, in the final step, we just "copy" the smallest image area from it, to the target canvas, which we produce the final image from.
